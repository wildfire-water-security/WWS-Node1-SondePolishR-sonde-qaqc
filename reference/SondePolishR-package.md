# SondePolishR: Perform Quality Assurance and Data Corrections on Sonde Data

Provides an interactive workflow for quality assurance and quality
control (QA/QC) of water quality sonde data. The package enables users
to import raw sonde datasets, review and correct observations, apply
quality flags, visualize time-series data alongside associated metadata,
summarize observations, and export cleaned datasets and metadata while
maintaining a reproducible record of all edits.
